# 🚀 Guía de Instalación y Ejecución Local
**Frontend Web SGCM (MedAgenda) - Next.js 14**

Sigue estos pasos para levantar el entorno de desarrollo del frontend en tu máquina local.

---

## 📋 Pre-requisitos

Antes de comenzar, asegúrate de tener instalado en tu sistema:
- **[Node.js](https://nodejs.org/es)** (Versión 18.17 o superior recomendada).
- **[Git](https://git-scm.com/)** para el control de versiones.
- Tener el proyecto Backend (API .NET 8) configurado y ejecutándose en tu máquina.

---

## ⚙️ Pasos de Instalación

### 1. Clonar el Repositorio
Abre tu terminal, clona el repositorio oficial y muévete a la rama específica del frontend web:
