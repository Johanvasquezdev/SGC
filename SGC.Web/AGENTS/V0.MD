# 🎨 v0 - Asistente Generador de Componentes UI

**v0 de Vercel** es una IA puramente especializada en crear pedazos de Interfaz de Usuario (UI) funcionales usando React, Tailwind y shadcn/ui. 

No pidas a v0 que haga APIs ni rutas. Pídele **piezas aisladas**, copiándolas luego a tu carpeta `src/components/`.

---

## 📋 Tips Rápidos para usar v0

Cuando pidas algo en [v0.dev](https://v0.dev), usa prompts como los siguientes para garantizar consistencia con el ecosistema moderno:

### Prompt: Dashboard del Paciente
> "A modern patient dashboard for a medical clinic. Include a summary section showing upcoming appointments with standard health icons. Add a sidebar on the left and a top header with the patient's avatar. Use Tailwind CSS and Shadcn UI components. Ensure it's responsive and feels premium with smooth hover effects and a clean blue-white color palette."

### Prompt: Tarjeta de Médico (`MedicoCard`)
> "A very aesthetic doctor profile card. Show an avatar image of the doctor, their specialization (e.g., Cardiology), a 5-star rating, and available hours badges. Include a primary button 'Agendar Cita' and a secondary button 'Ver Perfil'. Implement this as an encapsulated React Server component with Tailwind CSS. Make it look professional and trustworthy."

### Prompt: Formulario de Registro
> "Create a 3-step modern registration form for patients. Step 1: Personal info, Step 2: Contact info, Step 3: Password creation. Use a clean, distraction-free container with subtle shadows. It must look very modern and use Shadcn UI inputs, labels, and buttons."

### Prompt: Estadísticas Administrativas
> "Generate an admin dashboard grid containing 4 modern statistic cards (Total Patients, Upcoming Appointments, Active Doctors, Revenue). Below the cards, add a fake data table showing recent transactions. Use standard clear Tailwind utility classes and Lucide React icons."

---
**Estrategia:** Genera la pieza visual en v0 > inspecciona el código generado > cópialo y pégalo en tus componentes (`src/components/...`) > pídele a **Gemini** que los conecte a la página > pídele a **Claude** que les conecte los datos reales de la API.
